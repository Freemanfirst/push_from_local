# 2019.8.2

## 问题：电脑卡住，显示内存不足，无法打开应用。直接关机后，显示initramfs提示符进不了系统

## 解决：

1. 使用USB启动盘进入系统
2. 找到坏掉的Ubuntu分区
3. 执行 sudo fsck /dev/sda1（按实际）之后一路y 修复
4. 重启系统

# 2019.8.27

## 问题：笔记打不开，莫名被上锁，重启后，文件系统损坏

## 解决：

1. 使用USB启动盘进入系统
2. 找到坏掉的Ubuntu分区
3. 执行 sudo fsck /dev/sda1（按实际）之后一路y 修复
4. 重启系统

---

## 上面的问题多次发生，可能是内存太小的问题。加了内存条后还未发生过相似的问题
